Building production-grade web applications and on-chain infrastructure for the agent economy, autonomous credit, cross-chain settlement, MCP tooling and verifiable on-chain intelligence.

## 🔭 Current Focus

**On-Chain Intelligence & Agentic Infrastructure**

- Autonomous agent execution and settlement
- Cross-chain payment routing and discovery
- Model Context Protocol (MCP) integration
- Verifiable execution and reputation systems
- Multi-provider wallet scoring and trend classification
- Smart money tracking and signal detection
- Data pipelines and real-time monitoring dashboards

---

##  Core Projects

**Aurex — Latest on-chain project**

- Credit-weighted signal &amp; capital allocation system
- Inter-agent settlement with x402 payment flows
- MCP-based logging and execution tracking
- Built and shipped on the X Layer ecosystem

**AetherCredit — Autonomous credit infrastructure for AI agents**

- Turns verifiable compute history into on-chain credit, reputation, and working capital
- Built for the machine economy on the 0G network
- AI-agent native, with MCP integration
- TypeScript + Solidity smart contracts

**OMEN Intelligence — Multi-agent on-chain intelligence platform**

- Built on the Swarms Framework
- Serves memecoin traders and autonomous AI agents across Solana and BSC
- Multi-agent analysis pipeline for lifecycle and signal detection

**Short-Squeeze-Intelligence — Squeeze signal data pipeline**

- Built for Context Protocol
- Aggregates and processes market data to surface short-squeeze setups
- Real-time data pipeline feeding actionable signals

**Onchain Wallet Intelligence API v2 — Multi-provider wallet scoring**

- Auto multi-provider parallel fetching (Covalent + Alchemy + Etherscan)
- Wallet scoring (0–100) with trend classification utilities
- Pure dashboard frontend — summary cards, insights panel, wallet table
- Backend on Render, frontend (Next.js/TypeScript) on Vercel

**FlashBot Mint PRO — MEV-speed NFT minting bot**

- Multi-wallet minting controlled via Telegram
- Flashbots + 4 relay support (Titan, Beaverbuild, Rsync, BuildAI)
- ethers v6 with a nonce manager to prevent tx conflicts
- Deployed on Render

**Sniper Wallet Tracker — Real-time smart money monitoring**

- Monitors 64 sniper wallets via Alchemy WebSocket (ERC-721 Transfer events)
- Copy-mint confidence scoring
- Live alerting pipeline, deployed on Render

**Advanced Weather App — Real-time weather**

- React-powered, fetches live weather data by city
- Temperature, conditions, and forecasts in a sleek UI

**Discord UI Clone — Pixel-perfect frontend**

- Faithful clone of the Discord interface
- Demonstrates advanced layout and component architecture
